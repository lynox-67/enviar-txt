Paste your loadstring below. We'll wrap it with the environment logger so every API call is captured to logged.txt.

How to use

  1. 1 Execute the generated loadstring in your executor.
  2. 2 Let it run for 2–10 minutes so the logger can capture API calls.
  3. 3 Open your executor's workspace folder.
  4. 4 Locate the file named logged.txt.
  5. 5 Copy the entire contents of logged.txt.
  6. 6 Paste it into an AI model (ChatGPT, Claude, Gemini, Grok, etc.) and ask it to reconstruct the original script from the logged environment calls.
Tip: The longer you let the script run, the more complete the log will be — giving the AI more context to accurately reconstruct the source.
